## üìù CATATAN PENTING

### Faktor yang Mempengaruhi Harga:
1. **Kompleksitas desain** - Custom vs template
2. **Jumlah halaman/fitur** - Lebih banyak = lebih mahal
3. **Integrasi** - Payment gateway, API pihak ketiga
4. **Timeline** - Rush order = markup 20-50%
5. **Revisi** - Tentukan limit revisi di awal
6. **Konten** - Siapa yang menyiapkan

### Tips Negosiasi:
- Selalu mulai dari harga tertinggi
- Buat 3 paket (Basic, Standard, Premium)
- Client biasanya pilih yang tengah
- Jangan terlalu murah, nanti tidak dihargai
- Minta DP minimal 50%

### Dokumen Pendukung:
- Form Requirement (sudah dibuat)
- Proposal Template
- Quotation Template
- Kontrak/MOU
